Front-end

A front-end developer is responsible for th clien-side of a web application, which include the design of layout and interactive of the website or app. Front-end developer typically work with HTML, CSS, and JavaScript to create the user interface and user experience for a website or app. they may also work with frameworks like React or angular, and libraries like jquery and Bootstrap.

Html/Css & Js

HTML, CSS, and JavaScript are the three core technologies used in front-end web development. HTML stand for HyperText Markup Language, and it is used to create the structure of a website. CSS stand for Cascading Style Sheets, is used to style and format the HTML content. And JavaScript is a programming language used to add interactivity and functionality to a website.

Back-end

A Back-end developer is responsible for the server-side of a web application, which includes the structure, data, and functionality of the website or app. A Back-end developer typicallly works with server-side scripting language like PHP, Python, or Ruby, and databases like MySQL or MongoDB. they may also work with frameworks like Laravel or Django.
